Projects

NP Mario Kart - For this project I had to create 4 different mario kart maps inside of Python. In this I had to create “mushroom skips” , this was an NP Problem to determine the amount of skips the user can make by using a mushroom to take shortcuts in the level. (Code - https://github.com/KidAtum/GroupProject3Kart )

Mask Mayhem - This was a 3D Video Game made in Unity using C#. The premise of the game was to eliminate people that were not wearing masks indoors. The map was a replica of one of the buildings here at my campus. (Code - https://github.com/KidAtum/MaskMayhemLucas )

SHU Lost and Found - Me and a team of IT Professionals recreated the main homepage for my university's website. I particularly made the Lost and Found area. This is where students who lost a wallet, or keys would type in what they lost, and where they last saw the item. If someone found the item, they can contact the owner and then mark the lost item as found. (Can't give out code - Universities)

Grapple Racers - Grapple Racers is a 3D Parkour Racing game I created in Unity using C#. The premise of this game is to complete the parkour map within a certain amount of time. To complete the map, you would have to slide under obstacles, wall run so you don't fall down, grapple hook swing to different obstacles, or vault up ledges. This game is not 100% complete, but I have created one fully working map with all of the above obstacles. (Code not given, still finishing up)

Minting Dapp - I created a NFT Minting Dapp. The website itself was made in Visual Studio Code and coded using Javascript, HTML, and CSS. The contract for the NFTs and Minting was made in Remix.Ethereum using the Solidity language. This minting dapp was created to host my collection of NFTs live, so that others can go on the site and mint my NFTs. Each wallet was only allowed to mint up to 20, and the wallet had to be connected to ethereums blockchain. (Code not given - Risk of minting failures if people figure out the code)

Remade Space Invaders - I fully remade an exact replica of the classic game Space Invaders using Unity 2D and C#. The game had small aliens that were easy to eliminate, and of course a final boss fight which took 50 shots to eliminate. The game has a scoreboard, and a point system as well.
